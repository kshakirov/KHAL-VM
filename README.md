# KHAL-VM (Kirill + HAL Virtual Machine)

Исследовательская функциональная виртуальная машина с регистровой архитектурой и однопроходным Лисп-фронтендом. Проект создается итеративно, мелкими шагами, на стыке низкоуровневого системного программирования и теории функциональных языков.

---

## 🧭 СТРАТЕГИЯ РАЗВЕРТЫВАНИЯ БОРТА

1. **Этап 1 (Sandbox):** Проектирование физики ядра, системы команд (ISA) и синтаксиса в интерактивной среде Emacs. Прототипирование на скриптовых движках без инфраструктурного шума (`F#` / `Chez Scheme`).
2. **Этап 2 (Перенос на Металл):** Полный перенос проверенной логики на **чистый C++** по заветам **Бьёрна Страуструпа** (контроль ресурсов через RAII) и **Александра Степанова** (математическая строгость, обобщенные алгоритмы).

## 🚀 БЫСТРЫЙ СТАРТ (SANDBOX)

Проект поддерживает два альтернативных легковесных движка для запуска прямо из буфера Emacs:

* **F# скрипт:** `dotnet fsi src/khal_vm.fsx`
* **Chez Scheme скрипт:** `scheme --script src/khal_vm.ss`
